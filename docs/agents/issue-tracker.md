# Issue tracker: GitHub

Issues and PRDs for this repo live as GitHub issues in `xenoninja/guess-the-x`. Use the `gh` CLI for all operations.

## Conventions

- Create, read, comment on, label, and close issues using `gh issue`.
- Infer the repository from `git remote -v`. The configured `origin` is `git@github.com:xenoninja/guess-the-x.git`.
- When a skill says "publish to the issue tracker", create a GitHub issue.
- When a skill says "fetch the relevant ticket", run `gh issue view <number> --comments`.
